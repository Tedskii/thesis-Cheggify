How to use?

1
Place eggs to the capture Chamber

Follow the orientation of the holder when placing the eggs

2
Slide in the drawer to the imaging area

To start the classification, the system must first detect 30 eggs on its sight

3
Automatic analysis will start

A result snapshot will be presented in the screen of the size of each eggs.

4
Select the size of egg you want to classify.

It will engage the physical mechanism to classify the eggs based on user input

5
Slide out the drawer to remove the classified eggs.

A fresh batch of eggs can now be placed into the emptied slots to start the process again!
